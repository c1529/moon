numpy提供两个基本对象：ndarray(是存储单一数据类型的多维数组)和ufunc(对数组进行处理的函数)
## 创建一维数组
    import numpy as np
    arr1=np.array([1,2,3,4])
    print(arr1)
## 创建二维数组
    import numpy as np
    arr2=np.array([[1,2,3,4],[4,5,6,7]]) ** 二维数组两个【】 **
    print(arr2)
